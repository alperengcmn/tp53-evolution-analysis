# TP53 Comparative Evolution Analysis

This project analyzes the TP53 gene sequence of the Toros Blind Mole Rat and Human.

The TP53 gene is known for its role in tumor suppression and DNA repair.

Blind mole rats have remarkable cancer resistance mechanisms, making them interesting for comparative genomic studies.

## Pipeline

1 Download sequences from NCBI  
2 Perform FASTA quality control  
3 Calculate sequence statistics  
4 Perform pairwise alignment  
5 Evaluate sequence similarity

## Dataset

Human TP53 gene  
Toros Blind Mole Rat TP53 gene

## Tools

Python  
BioPython  
NCBI database
FASTA parsing using Python
